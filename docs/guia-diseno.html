🔤 Tipografía

Fuentes principales utilizadas en el diseño:

1. Neo Sans Pro / Neo Sans Pro Ultra

Uso: Headings principales (H1, H2), Textos destacados

Peso: Regular (400), Bold (700), Ultra (900)

font-family: 'Neo Sans Pro Ultra', 'Neo Sans Pro', sans-serif;

2. Source Sans Pro

Uso: Headings secundarios, Textos corporativos

Peso: Regular (400), Semibold (600), Bold (700)

font-family: 'Source Sans Pro', Arial, sans-serif;

3. Roboto

Uso: Body text, Párrafos, Contenido general

Peso: Light (300), Regular (400), Medium (500), Bold (700)

font-family: 'Roboto', sans-serif;

4. Montserrat

Uso: Botones, Labels, Textos destacados alternativos

Peso: Regular (400) - ExtraBold (900)

font-family: 'Montserrat', sans-serif;

5. Karla

Uso: Textos complementarios, Alternativas

font-family: 'Karla', sans-serif;

💻 Ejemplos de Uso

1. Importar el CSS:

<link rel="stylesheet" href="mediosconvalor-design-system.css">

2. Usar Variables CSS:

.mi-elemento {
  color: var(--color-text-white);
  background-color: var(--color-dark-bg);
  font-family: var(--font-body);
  font-size: var(--font-size-body);
}

3. Hero Section Completa:

<section class="hero">
  <div class="container">
    <h1>NO SOLO DECIMOS, LO HACEMOS</h1>
    <h2>SERVICIOS</h2>
    <p>Descripción del servicio...</p>
    <button class="btn-primary">CONTACTO</button>
  </div>
</section>

📝 Notas Importantes

  • Contraste: El verde primario (#00DD2A) sobre fondo oscuro proporciona excelente contraste y visibilidad.
  • Modo Oscuro: El diseño está optimizado para fondo oscuro, esto es su identidad visual principal.
  • Sostenibilidad Digital: Los tonos oscuros reducen consumo de energía en pantallas modernas.
  • Animaciones: Las transiciones están configuradas a 0.3s para una experiencia suave sin resultar lenta.
  • Flexibilidad: Todos los valores están en variables CSS para fácil mantenimiento y personalización.
  • Accesibilidad: Se utilizan colores con suficiente contraste según WCAG 2.1 AA.
